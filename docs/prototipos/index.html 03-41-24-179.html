Mapeamento do fluxo de autenticação

Base visual para Figma. Este arquivo não é tela final do sistema: ele serve para mapear as etapas, os campos e a navegação ideal entre login, cadastro e redefinição de senha.

1. Login 2. Esqueci minha senha 3. Nova senha 4. Confirmação 5. Cadastro inicial

Tela 01

Login

Entrada principal do sistema para usuário já cadastrado.

Validações esperadas

  • E-mail obrigatório e com formato válido.
  • Senha obrigatória.
  • Mensagem de erro clara em credenciais inválidas.
  • Ao logar com sucesso, redireciona para o sistema.

Tela 02

Esqueci minha senha

Fluxo inicial para recuperação segura de acesso.

Logo / Marca
Enviar link ou código de redefinição apenas para e-mails válidos.
Voltar para login

Regras de segurança

  • Não revelar se o e-mail existe ou não no sistema.
  • Mensagem neutra: “Se o e-mail estiver cadastrado, enviaremos as instruções”.
  • Token de redefinição com expiração.

Tela 03

Redefinir senha

Tela aberta após clique no link seguro de recuperação.

Token válido para redefinição
Mínimo sugerido: 8 caracteres, com letras e números.
As senhas devem coincidir.

Validações esperadas

  • Token válido e não expirado.
  • Nova senha obrigatória.
  • Confirmação obrigatória e igual à nova senha.
  • Ao finalizar, direcionar para confirmação.

Tela 04

Confirmação de redefinição

Tela de sucesso após atualização segura da senha.

Sua senha foi atualizada. Agora você já pode voltar para a tela de login e acessar o sistema com as novas credenciais.

Objetivo da tela

  • Fechar o fluxo com feedback claro.
  • Evitar deixar o usuário perdido após trocar a senha.

Tela 05

Cadastro inicial

Fluxo separado do login, usando apenas os campos mínimos definidos no projeto.

Logo / Marca
Já possui cadastro? Ir para login

Campos do cadastro inicial

  • Nome, e-mail, senha e cargo.
  • Status, id e datas são preenchidos pelo sistema.
  • Dados complementares ficam para perfil/configurações depois.

Resumo do fluxo

  • Usuário entra pela tela de login.
  • Se esqueceu a senha, vai para recuperação e depois redefine a senha em tela segura.
  • Após redefinir, recebe uma confirmação e volta ao login.
  • Se ainda não possui conta, vai para cadastro inicial.
  • O cadastro pede apenas os dados mínimos; perfil completo fica para etapa posterior.
